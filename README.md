# Quasar: Beam Your Files

## About Quasar
Quasar is a terminal-based [P2P](https://en.wikipedia.org/wiki/Peer-to-peer) file sharing utility built for [Hack Club Stardance](https://stardance.hackclub.com).

## Why Quasar?
- Secure (uses [magic wormholes](https://en.wikipedia.org/wiki/Magic_Wormhole) to transfer your files)
- Written in the Go programming language
    - Fast & lightweight
    - Cross-platform binaries
